## v1.1.0 — 2026-04-05

### 機能追加

- **自動保存（常時）** (`src/components/MatchForm.tsx`)
  - 得点入力・取消時に常に自動保存されるよう拡張（ライブ中以外でも）
  - 自動保存成功時に「保存済み ✓」トースト表示（2秒で消える）
  - `handleRecordGoal` と `handleUndoGoal` の `isLive` 条件を削除

- **保存ボタン整理** (`src/components/MatchForm.tsx`)
  - 「途中保存 (現在のスコアを公開)」ボタンを削除（自動保存と重複するため）
  - `isLive=true` 時のボタンラベルを「試合終了して保存」に変更（動作を明確化）

- **得点ランキング表示機能強化** 
  - 新規ユーティリティ `src/lib/scoring.ts` を作成（`calcTopScorers` 関数）
    - 試合データから得点ランキングを計算する共通関数
    - `playerNames` でフィルタ、`limit` で件数制限可能
  - ダッシュボード試合履歴タブに「TOP SCORERS」セクション追加 (`src/app/grade/[gradeId]/page.tsx`)
    - 上位5名をカード形式で表示（スマホに最適化）
  - 選手管理ページのランキング計算を `scoring.ts` に移行 (`src/app/grade/[gradeId]/players/page.tsx`)

- **AI試合分析機能** (Gemini API 無料枠使用)
  - 新規API: `src/app/api/matches/analyze/route.ts`
    - 試合データを受け取り Gemini 1.5 Flash で分析
    - 試合の振り返り・良かった点・改善点を日本語（150～200字）で生成
    - 事前に GEMINI_API_KEY を Vercel 環境変数に設定が必要
  - 新規コンポーネント `src/components/MatchAnalysis.tsx` (Client Component)
    - 試合詳細ページに「AI で試合を分析」ボタンを配置
    - クリックで分析結果をカード内に表示
  - 試合詳細ページに統合 (`src/app/grade/[gradeId]/match/[matchId]/view/page.tsx`)

### 依存パッケージ追加
- `@google/generative-ai` 版本は npm install で自動取得

## v1.0.0 — 2026-03-15

### UI / デザイン
- **MatchList: 完了試合をリスト形式に変更**
  - 完了試合：コンパクトなリスト行（結果バッジ + スコア + 対戦相手 + 得点者 + 日付 + 形式）
  - ライブ試合：従来のカード形式を維持
- **結果ラベルを変更**: VICTORY/DEFEAT → WIN/LOSE
- **スコア間の「-」をスコアと同色**（slate-300 → slate-900）
- **試合形式の表示を改善**:
  - "Halves" → "前後半"
  - "15Mマッチ" → 実際の `matchDuration` を使用（例：`8min`）
- **得点者を折り返し全表示**（truncate削除 → break-all）
- **テキストコントラスト改善**（slate-200/300 → slate-400/500）
- **屋外モード機能を削除**（Header.tsx・globals.css から完全除去）

### 機能追加
- **アーカイブページ** (`/grade/[gradeId]/archive`)
  - 直近以外の試合を月別グループで表示
  - 月ごとのW/L/D集計を表示
  - 試合履歴画面に「過去の試合を見る」リンクを追加
- **アクティブ月フィルター**: 試合履歴は当月＋前月（15日以前の場合）のみ表示

### セキュリティ
- **middleware.ts 新規作成**: 未ログイン時は編集ページ・新規入力ページをサインインへリダイレクト
  - 保護対象: `/grade/*/match/new`, `/grade/*/match/[matchId]`
- **ダッシュボードのタブ制御**: 「試合入力」「選手管理」タブをログイン済みのみ表示
- **読み取り専用の試合詳細ページ** (`/grade/[gradeId]/match/[matchId]/view`) 新規作成
  - 誰でも閲覧可能
  - ログイン済みユーザーのみ「編集する」ボタンを表示
  - MatchList・アーカイブページのリンクをすべて `/view` に変更

### マニュアル更新 (`/manual`)
- 画面構成（タブ3種）の説明を追加
- 得点記録フロー（選手選択→ボタン保存）の説明を追加
- アーカイブページの説明を追加
- 「簡易/通常入力モード」の記述を実際のUIに合わせて整理
- セクション番号を05まで拡張
